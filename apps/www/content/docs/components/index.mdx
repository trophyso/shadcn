## Installation

Install components using the shadcn CLI:

```bash
npx shadcn@latest add https://ui.trophy.so/r/<component-name>.json
```

Or install all components:

```bash
npx shadcn@latest add https://ui.trophy.so/r/all.json
```

## All Components

### Streaks
Components for displaying and managing user streaks.
- [Streak Badge](/docs/components/streak-badge) - Display current streak count
- [Streak Calendar](/docs/components/streak-calendar) - Visual calendar of streak history
- [Streak Card](/docs/components/streak-card) - Composed streak summary with calendar and stats

### Achievements
Components for badges, progress, and unlock celebrations.
- [Achievement Badge](/docs/components/achievement-badge) - Single achievement card
- [Achievement Grid](/docs/components/achievement-grid) - Grid of multiple achievements
- [Achievement List](/docs/components/achievement-list) - Vertical list layout with progress
- [Achievement Unlocked](/docs/components/achievement-unlocked) - Celebration modal
- [Achievement Card](/docs/components/achievement-card) - Composed achievements overview card

### Leaderboards
Components for rankings and competition.
- [Leaderboard Rankings](/docs/components/leaderboard-rankings) - Ranked user list with avatars, bylines, totals, and top-3 crowns
- [Leaderboard Podium](/docs/components/leaderboard-podium) - Top 3 visual display
- [Leaderboard Card](/docs/components/leaderboard-card) - Composed card with leaderboard date range, podium, and rankings list

### Points
Components for point displays.
- [Points Badge](/docs/components/points-badge) - Show total points
- [Points Chart](/docs/components/points-chart) - Visualize points over time
- [Points Awards](/docs/components/points-awards) - Show point award events by date and action
- [Points Levels](/docs/components/points-levels) - Show a vertical levels timeline with optional details and progress
- [Points Levels (Simple)](/docs/components/points-levels-simple) - Show available points tiers and current level in a compact table
- [Points Boost](/docs/components/points-boost) - Promote temporary points multipliers with CTA and countdown
