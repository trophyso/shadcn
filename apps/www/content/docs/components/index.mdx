# Components

Trophy UI provides pre-built React components for gamification features. Each component is designed to work with the [Trophy API](https://docs.trophy.so) and follows shadcn/ui patterns.

## Installation

Install components using the shadcn CLI:

```bash
npx shadcn@latest add https://ui.trophy.so/r/<component-name>.json
```

Or install all components:

```bash
npx shadcn@latest add https://ui.trophy.so/r/all.json
```

## Categories

### Streaks
Components for displaying and managing user streaks.
- [Streak Badge](/docs/components/streak-badge) - Display current streak count
- [Streak Calendar](/docs/components/streak-calendar) - Visual calendar of streak history
- [Streak Freeze Indicator](/docs/components/streak-freeze-indicator) - Show available freezes
- [Streak At Risk](/docs/components/streak-at-risk) - Alert when streak is expiring

### Achievements
Components for badges, progress, and unlock celebrations.
- [Achievement Badge](/docs/components/achievement-badge) - Display achievement status
- [Achievement Grid](/docs/components/achievement-grid) - Grid of multiple achievements
- [Achievement Unlocked](/docs/components/achievement-unlocked) - Celebration modal
- [Achievement Progress](/docs/components/achievement-progress) - Progress toward achievement

### Leaderboards
Components for rankings and competition.
- [Leaderboard](/docs/components/leaderboard) - Full leaderboard display
- [Leaderboard Entry](/docs/components/leaderboard-entry) - Single ranking row
- [Leaderboard Podium](/docs/components/leaderboard-podium) - Top 3 visual display
- [User Rank](/docs/components/user-rank) - Current user's ranking card

### Points
Components for point displays and animations.
- [Points Display](/docs/components/points-display) - Show total points
- [Points Animation](/docs/components/points-animation) - Floating point animations

### Core
Provider and utilities.
- [Trophy Provider](/docs/components/trophy-provider) - Context provider for Trophy config
