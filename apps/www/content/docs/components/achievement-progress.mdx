# Achievement Progress

A progress bar component designed for metric-based achievements. Shows visual progress toward a goal with optional labels and percentages.

<ComponentPreview name="achievement-progress" />

## Installation

```bash
npx shadcn@latest add https://ui.trophy.so/r/achievement-progress.json
```

## Usage

Import the component:

```tsx
import { AchievementProgress } from "@/components/ui/achievement-progress"
```

Basic usage:

```tsx
<AchievementProgress
  achievement={{
    id: "1",
    name: "Task Master",
    metricValue: 100,
  }}
  currentValue={75}
/>
```

With Trophy API data:

```tsx
const achievement = await trophy.achievements.get({ 
  userId: "user_123",
  achievementId: "task-master"
})

<AchievementProgress
  achievement={achievement}
  currentValue={userProgress}
/>
```

## Examples

### With Percentage

```tsx
<AchievementProgress
  achievement={achievement}
  currentValue={75}
  showPercentage
/>
```

### With Values

```tsx
<AchievementProgress
  achievement={achievement}
  currentValue={75}
  showValue
/>
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `achievement` | `Achievement` | Required | Achievement data with metricValue |
| `currentValue` | `number` | Required | Current progress value |
| `showPercentage` | `boolean` | `false` | Show percentage label |
| `showValue` | `boolean` | `false` | Show current/target values |
| `showName` | `boolean` | `true` | Show achievement name |
| `animate` | `boolean` | `true` | Animate the progress bar |

### Types

```ts
interface Achievement {
  id: string
  name: string
  metricValue?: number
}
```

## Accessibility

- Uses proper progress element semantics
