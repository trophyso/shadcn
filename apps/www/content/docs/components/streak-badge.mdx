## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `streak` | `StreakResponse` | - | Streak data object |
| `length` | `number` | - | Manual streak length (alternative to streak prop) |
| `size` | `"sm" \| "default" \| "lg"` | `"default"` | Badge size |
| `showFrequency` | `boolean` | `false` | Show frequency label (e.g., "7 days") |
| `showFlame` | `boolean` | `true` | Show the flame icon |
| `icon` | `ReactNode` | - | Custom icon to replace flame |

### Types

```ts
interface StreakResponse {
  length: number
  frequency: "daily" | "weekly" | "monthly"
}
```
