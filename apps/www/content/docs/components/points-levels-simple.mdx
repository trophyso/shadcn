## With Trophy

Use the [Trophy SDK](https://docs.trophy.so/api-reference/client-libraries) to fetch user progress and system levels server-side:

<OpenApiCodeSamples spec="application.yml" method="get" endpoint="/users/{id}/points/{key}" />

Use the [Trophy SDK](https://docs.trophy.so/api-reference/client-libraries) to fetch points system level definitions server-side:

<OpenApiCodeSamples spec="application.yml" method="get" endpoint="/points/{key}/levels" />

Then aggregate both responses and pass them into `PointsLevelsSimple` props in your UI layer:

```tsx
const currentPoints = userPointsResponse.total

const levels = (pointsLevelsResponse.levels ?? []).map((level) => ({
  id: level.key,
  name: level.name,
  threshold: level.points,
  iconType: level.key,
}))

<PointsLevelsSimple
  levels={levels}
  currentPoints={currentPoints}
  showProgressBar
/>
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `levels` | `PointsLevelSimple[]` | Required | Ordered level definitions used to render thresholds and labels |
| `currentPoints` | `number` | - | User points total used to determine current level and progress |
| `showProgressBar` | `boolean` | `false` | Shows a progress header toward the next level when `currentPoints` is available |
| `currentLevelLabel` | `string` | `"Your current level"` | Label shown on the active row |
| `formatPoints` | `(value: number) => string` | - | Optional formatter for threshold and progress values |

### Types

```ts
type PointsLevelSimpleIconType =
  | "beginner"
  | "novice"
  | "intermediate"
  | "professional"
  | "expert"
  | "master"
  | "grand-master"
  | "enlightened"

interface PointsSubLevelSimple {
  name: string
  threshold: number
}

interface PointsLevelSimple {
  id: string
  threshold: number
  name: string
  iconType?: PointsLevelSimpleIconType
  subLevels?: PointsSubLevelSimple[]
}
```
