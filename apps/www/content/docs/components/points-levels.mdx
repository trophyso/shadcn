## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `levels` | `PointsLevel[]` | Required | Vertical list of levels |
| `currentPoints` | `number` | - | User total used to derive current level and progress |
| `currentLevelLabel` | `string` | `"Current"` | Label shown next to the current level |
| `formatPoints` | `(value: number) => string` | - | Optional formatter used when rendering computed ranges |

### Types

```ts
interface PointsSubLevel {
  name: string
  threshold: number
}

interface PointsLevel {
  id: string
  name: string
  threshold: number
  summary?: string
  subLevels?: PointsSubLevel[]
}
```
