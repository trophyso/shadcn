## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `levels` | `PointsLevel[]` | Required | List of levels to display |
| `currentLevelId` | `string` | - | Level `id` that should display the current level label |
| `currentPoints` | `number` | - | Current points value used to render progress toward the next level when `currentLevelId` is set |
| `currentLevelLabel` | `string` | `"Your current level"` | Label displayed for the row matching `currentLevelId` |
| `formatPoints` | `(value: number) => string` | - | Optional formatter for range values |

### Types
