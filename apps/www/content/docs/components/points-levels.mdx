## With Trophy

Use the [Trophy SDK](https://docs.trophy.so/api-reference/client-libraries) to fetch user progress and system levels server-side:

<OpenApiCodeSamples spec="application.yml" method="get" endpoint="/users/{id}/points/{key}" />

Use the [Trophy SDK](https://docs.trophy.so/api-reference/client-libraries) to fetch points system level definitions server-side:

<OpenApiCodeSamples spec="application.yml" method="get" endpoint="/points/{key}/levels" />

Then aggregate both responses and pass them into `PointsLevels` props in your UI layer:

```tsx
const currentPoints = userPointsResponse.total

const levels = (pointsLevelsResponse.levels ?? []).map((level) => ({
  id: level.key,
  name: level.name,
  threshold: level.points,
  summary: level.description,
  subLevels: level.subLevels?.map((subLevel) => ({
    name: subLevel.name,
    threshold: subLevel.points,
  })),
}))

<PointsLevels levels={levels} currentPoints={currentPoints} />
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `levels` | `PointsLevel[]` | Required | Vertical list of levels |
| `currentPoints` | `number` | - | User total used to derive current level and progress |
| `currentLevelLabel` | `string` | `"Current"` | Label shown next to the current level |
| `formatPoints` | `(value: number) => string` | - | Optional formatter used when rendering computed ranges |

### Types

```ts
interface PointsSubLevel {
  name: string
  threshold: number
}

interface PointsLevel {
  id: string
  name: string
  threshold: number
  summary?: string
  subLevels?: PointsSubLevel[]
}
```
