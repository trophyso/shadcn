## Usage

Import the component:

```tsx
import { PointsBadge } from "@/components/ui/points-badge"
```

## Examples

### Basic Usage

<ComponentPreview name="points-badge" />

```tsx
<PointsBadge
  points={{ name: "XP", total: 2500, badgeUrl: null }}
/>
```

### Sizes

Available in three sizes: `sm`, `default`, and `lg`.

<ComponentPreview name="points-badge-sizes" />

```tsx
<PointsBadge points={points} size="sm" />
<PointsBadge points={points} size="default" />
<PointsBadge points={points} size="lg" />
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `points` | `PointsBadgeData` | Required | Points data object |
| `size` | `"sm" \| "default" \| "lg"` | `"default"` | Component size |
| `icon` | `ComponentType` | - | Custom icon component when no badge image is available |
| `formatValue` | `(value: number) => string` | - | Custom value formatter |

### Types

```ts
interface PointsBadgeData {
  name: string
  badgeUrl: string | null
  total: number
}
```
