## With Trophy

Use the [Trophy SDK](https://docs.trophy.so/api-reference/client-libraries) to fetch user progress server-side:

<OpenApiCodeSamples
  spec="application.yml"
  method="get"
  endpoint="/users/{id}/points/{key}"
/>

Use the [Trophy SDK](https://docs.trophy.so/api-reference/client-libraries) to fetch points system level definitions server-side:

<OpenApiCodeSamples
  spec="application.yml"
  method="get"
  endpoint="/points/{key}/levels"
/>

Then aggregate both responses and pass them into `PointsLevelsTimeline` props in your UI layer. Trophy does not support sub-levels yet.

```tsx
const currentPoints = userPointsResponse.total

const levels = (pointsLevelsResponse.levels ?? []).map((level) => ({
  ...level,
  subLevels: [],
}))

<PointsLevelsTimeline levels={levels} currentPoints={currentPoints} />
```

## API Reference

### Props

| Prop                | Type                        | Default     | Description                                            |
| ------------------- | --------------------------- | ----------- | ------------------------------------------------------ |
| `levels`            | `PointsLevelTimeline[]`     | Required    | Vertical list of levels                                |
| `currentPoints`     | `number`                    | -           | User total used to derive current level and progress   |
| `currentLevelLabel` | `string`                    | `"Current"` | Label shown next to the current level                  |
| `formatPoints`      | `(value: number) => string` | -           | Optional formatter used when rendering computed ranges |

### Types

```ts
interface PointsSubLevelTimeline {
  name: string
  points: number
}

interface PointsLevelTimeline {
  id: string
  key?: string
  name: string
  description?: string
  badgeUrl?: string | null
  points: number
  subLevels?: PointsSubLevelTimeline[]
}
```
