## Usage

Import the component:

```tsx
import { StreakAtRisk } from "@/components/ui/streak-at-risk"
```

With Trophy API data:

```tsx
const streak = await trophy.streaks.get({ userId: "user_123" })

{streak.expires && (
  <StreakAtRisk streak={streak} />
)}
```

## Examples

### With Countdown

Show a live countdown timer.

```tsx
<StreakAtRisk streak={streak} showCountdown />
```

### Dismissible

Allow users to dismiss the alert.

```tsx
<StreakAtRisk 
  streak={streak} 
  dismissible 
  onDismiss={() => setDismissed(true)} 
/>
```

### Custom Thresholds

```tsx
<StreakAtRisk 
  streak={streak} 
  warningHours={8} 
  criticalHours={3} 
/>
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `streak` | `StreakResponse` | Required | Trophy API streak response |
| `warningHours` | `number` | `6` | Hours threshold for warning state |
| `criticalHours` | `number` | `2` | Hours threshold for critical state |
| `showCountdown` | `boolean` | `false` | Show countdown timer |
| `dismissible` | `boolean` | `false` | Allow dismissing the alert |
| `onDismiss` | `() => void` | - | Callback when dismissed |

### Types

```ts
interface StreakResponse {
  length: number
  frequency: "daily" | "weekly" | "monthly"
  expires: string | null
}
```

## Visual States

| State | Condition | Appearance |
|-------|-----------|------------|
| Warning | 2-6 hours remaining | Yellow/amber styling |
| Critical | < 2 hours remaining | Red styling |
| Hidden | > 6 hours remaining | Not displayed |

## Accessibility

- Uses `role="alert"` for important announcements
- Clear, actionable messaging
- Dismissible with keyboard
