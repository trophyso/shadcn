## With Trophy

Use the [Trophy SDK](https://docs.trophy.so/api-reference/client-libraries) to fetch the user's points data server-side:

<OpenApiCodeSamples spec="application.yml" method="get" endpoint="/users/{id}/points/{key}" />

Then aggregate and pass the response into `PointsAwards` props in your UI layer:

```tsx
const rows = (response.awards ?? []).reduce((acc, award) => {
  const date = award.date.slice(0, 10)
  const existing = acc.get(date) ?? {
    id: date,
    date,
    totalPoints: award.total,
    actions: [],
  }

  existing.actions.push({
    type: award.trigger.type,
    points: award.awarded,
    label: award.trigger.metricName,
  })

  acc.set(date, existing)
  return acc
}, new Map())

<PointsAwards rows={Array.from(rows.values())} />
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `rows` | `PointsAwardRow[]` | Required | Award history rows sorted however you prefer (often most recent first) |
| `formatTotalPoints` | `(value: number) => string` | - | Optional formatter for each row's total points value |
| `formatAwardedPoints` | `(value: number) => string` | - | Optional formatter for each row's awarded delta (sum of action points) |

### Types

```ts
type PointsAwardActionType =
  | "lesson-completed"
  | "quiz-passed"
  | "assignment-submitted"
  | "streak-maintained"
  | "discussion-contribution"
  | "flashcards-reviewed"

interface PointsAwardAction {
  type: PointsAwardActionType
  points: number
  label?: string
}

interface PointsAwardRow {
  id: string
  date: string
  totalPoints: number
  actions: PointsAwardAction[]
}
```
