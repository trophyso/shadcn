# Streak Calendar

A visual calendar component that displays streak activity history. Shows active days, freeze usage, and supports both Sunday and Monday week starts.

<ComponentPreview name="streak-calendar" />

## Installation

```bash
npx shadcn@latest add https://ui.trophy.so/r/streak-calendar.json
```

## Usage

Import the component:

```tsx
import { StreakCalendar } from "@/components/ui/streak-calendar"
```

With Trophy API data:

```tsx
const streak = await trophy.streaks.get({ userId: "user_123" })

<StreakCalendar streak={streak} />
```

## Examples

### Specific Month

```tsx
<StreakCalendar 
  streak={streak} 
  month={new Date(2024, 0, 1)}
/>
```

### Monday Start

```tsx
<StreakCalendar streak={streak} startOfWeek={1} />
```

### Without Freeze Indicators

```tsx
<StreakCalendar streak={streak} showFreezes={false} />
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `streak` | `StreakResponse` | Required | Trophy API streak response with history |
| `month` | `Date` | Current month | Month to display |
| `showFreezes` | `boolean` | `true` | Show freeze indicators on days |
| `startOfWeek` | `0 \| 1` | `0` | Week start: 0 = Sunday, 1 = Monday |
| `onDayClick` | `(date: Date, wasActive: boolean) => void` | - | Callback when day is clicked |

### Types

```ts
interface StreakResponse {
  streakHistory: StreakPeriod[]
}

interface StreakPeriod {
  periodStart: string
  periodEnd: string
  usedFreeze?: boolean
}
```

## Visual States

| State | Appearance |
|-------|------------|
| Active day | Orange background |
| Freeze used | Blue background with snowflake icon |
| Today | Ring outline |
| Future | Muted, disabled |
| Inactive | Default background |

## Accessibility

- Uses `role="grid"` with proper grid semantics
- Column headers with `role="columnheader"`
- Each day has descriptive `aria-label` with date and status
- Today marked with `aria-current="date"`
