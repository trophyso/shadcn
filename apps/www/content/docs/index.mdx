Trophy UI is a component library built on top of shadcn/ui to help you build gamification experiences faster. It provides pre-built components for streaks, achievements, leaderboards, points, and more.

Components are available via the shadcn CLI:

```bash
npx shadcn@latest add https://ui.trophy.so/r/<component>.json
```

For example, to install the Streak Badge component:

```bash
npx shadcn@latest add https://ui.trophy.so/r/streak-badge.json
```

## Component Categories

### Streaks

Track and display user streaks with beautiful, animated components:

- **Streak Badge** - Compact badge showing current streak count
- **Streak Calendar** - Monthly calendar view of streak history
- **Streak Freeze Indicator** - Shows available streak freezes
- **Streak At Risk** - Alert when streak is about to expire

### Achievements

Celebrate user accomplishments with customizable achievement displays:

- **Achievement Badge** - Individual achievement cards with locked/unlocked states
- **Achievement Grid** - Grid layout for multiple achievements
- **Achievement Unlocked** - Celebration animation for new achievements
- **Achievement Progress** - Progress tracking toward achievements

### Leaderboards

Rank users and drive competition with leaderboard components:

- **Leaderboard** - Full leaderboard with rankings
- **Leaderboard Entry** - Individual ranking row
- **Leaderboard Podium** - Top 3 podium display
- **User Rank** - Compact card showing user's position

### Points

Display and animate point totals:

- **Points Display** - Show point totals with icons
- **Points Animation** - Animated point changes
