# Trophy UI

Trophy UI provides pre-built, customizable React components specifically designed for gamification applications.

## Installation

```bash
npx shadcn@latest add https://ui.trophy.so/r/streak-badge.json
```

## Components

- **Streaks**: Display and track user streaks
- **Achievements**: Show badges and progress
- **Leaderboards**: Rank users and show standings
- **Points**: Display scores and animations
